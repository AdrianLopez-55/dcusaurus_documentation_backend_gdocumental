# Documents Type Schema

El schema `DocumentationSchema` representa la estructura de un tipo de documento la base de datos.

## Propiedades

| Nombre         | Tipo     | Requerido | Descripción                   |
| -------------- | -------- | --------- | ----------------------------- |
| TypeName       | string   | Sí        | Tipo de documento            |
