# Document Type Data Model

El modelo `DocumentationType` representa los datos de un tipo de documento en la base de datos.

## Propiedades

| Nombre         | Tipo     | Requerido | Descripción                   |
| -------------- | -------- | --------- | ----------------------------- |
| TypeName       | string   | Sí        | Tipo de documento            |
